¡Bienvenido al juego de Pasapalabra de
ISDI Coders!

Antes de empezar te explicaremos cómo se juega, cuáles son las reglas.

El juego consiste en:

- El jugador, con ayuda de preguntas, tendrá que adivinar una palabra relacionada con cada una de las letras del alfabeto.

- La palabra podra empezar con dicha letra o simplemente contener esta, pero traquilo que la propia pregunta lo indicará.

- Las palabras deberán ser escritas sin tildes ni dieresis, y en caso de que se puedan escribir en femenino o masculino, simempre en masculino.

- Si se acierta una palabra esta se guardará como acertada.

- Si se escribe 'Pasapalabra', se deja en blancola casilla, o se pulsa 'Cancelar' se pasará de turno y la pregunta se guardará para la siguiente ronda.

- Si se falla una pregunta la palabra se dará como fallada y no se guardará.

- Las rondas serán infinitas.

- Habrá un límite de tiempo de 300 segundos, y el juego se acabará cuando se acabe este.

- El juego podrá ser terminado en todo momento introduciendo la palabra "END", en caso de terminar antes de que acaben las preguntas el jugador recibirá la cantidad de palabras acertadas, pero no aparecerá en el ranking.

Si estás listo pulsa el botón de "jugar" y ¡empezamos!

El juego se llama PASAPALABRA. Consiste en adivinar 27 palabras, cada una de las cuales se corresponde con una letra del rosco. Para ello, se brinda una definición relativa al concepto de dicha palabra.


Cuanto mayor sea el número de palabras adivinadas, mayor será el puntaje obtenido al finalizar la partida. Se tiene un tiempo máximo de 150 segundos para responder a todas las preguntas. Las palabras deben ser introducidas sin tildes.


Para comenzar la partida, pulse START. Para finalizar la partida, pulse END. Para responder una pregunta, complete el campo y seleccione ENVIAR (no utilizar tildes). Para saltar la pregunta pulse PASAPALABRA.

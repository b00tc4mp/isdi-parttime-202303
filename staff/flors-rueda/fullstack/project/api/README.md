# Ballopolis Maze Riders: API
bloh bloh

## Technologies:

- Node & Express
- MongoDB & Mongoose
- Mocha Chai

## Install
```sh
npm i
```

## Run tests
- make sure to set the env variable of the database to test

```sh
npm run test
```

## Run test coverage
- make sure to set the env variable of the database to test

```sh
npm run test-coverage
```

## Run
```sh
npm run start
```

## Inspect
```sh
npm run inspect
```

## Watch
```sh
npm run watch
```
